# Tejas' Tech Journey - Final Project

Final project for Web Design I (AMEC 2050C), taught by Professor Eric Anderson.

## Live Website

Visit the site here: [https://ghodketejas.github.io/Final-Project/](https://ghodketejas.github.io/Final-Project/)

## About the Project

Tejas' Tech Journey is a beginner-friendly website focused on making computer hardware and technology easier to understand.  
The content is written from a learning perspective and is organized into clear, approachable sections.

## Main Sections

- **Home**: Introduction and site overview.
- **PC Basics**: Simple explanations of core computer components.
- **My Journey**: Personal learning experiences and milestones.
- **Tech Thoughts**: Opinions and reflections on tech setups and gear.

## Purpose

This project was created to:

- Practice web design principles from class.
- Build a multi-page personal website.
- Share tech knowledge in a way that feels less intimidating for beginners.

## Running Locally

To view the project on your machine, open `index.html` in a web browser.
